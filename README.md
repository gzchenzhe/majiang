【核心功能】

玩家库：维护常打牌的朋友列表（增删改查）
录入一场牌：选日期 → 勾选 4 个玩家 → 设置每个人坐的方位（东南西北）→ 输入每人这场的输赢金额（自动校验总和=0）
每日汇总：当日打了几场、我总输赢、其他人总输赢
每月汇总：本月总输赢、胜率、最大单场赢/输、连胜连败、月度趋势折线图

【分析功能】
5. 个人座位胜率：我坐东/南/西/北各方位时的胜率和平均输赢
6. 上下家分析：和不同人组合时（上家是谁/下家是谁/对家是谁），我的胜率和输赢
7. 对手分析：和每个朋友打过多少场、互相输赢统计
8. 智能座位推荐：输入今天的 3 个对手 → 输出"建议你坐 X 位，历史胜率 X%（基于 N 场数据）"，数据不足时给出提示
【旺位功能】
9. 今日财神方位：根据当日干支自动计算并显示（东/南/西/北/东南/西南/东北/西北）
10. 在录入界面显示"今日财位：东"，方便参考
【可视化】
11. 月度输赢折线图
12. 各方位胜率柱状图
13. 上下家关系热力图（可选）
【其他】
14. 数据存浏览器本地（LocalStorage / IndexedDB），不会丢
15. 支持导出 JSON 备份 / 从 JSON 恢复
16. 支持删除/修改最近一场录入错误的记录
