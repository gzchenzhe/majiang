# 麻将记分 · 功能介绍

一款纯网页的麻将记分工具，打开即用，数据存于手机本地，无需注册、无需网络（首次加载后即可离线使用）。iPhone 用户通过 Safari "添加到主屏幕"可像原生 App 一样全屏运行。

---

## 首页看板

打开即见今日概况：

- **今日干支与财神方位**：根据中国传统干支历自动计算当天的天干地支，并显示对应的财神方向（甲乙→东北、丙丁→正西、戊己→正北、庚辛→正东、壬癸→正南）。如果和自家黄历对不上，可在"我的"页面校准基准日期。
- **本月汇总**：当月累计输赢金额、已打场数、胜率、最佳一场的收益，一眼看清本月手气。

首页下方会列出今日已录入的每一场记录，点击可查看详情或编辑。

---

## 录入一场牌

这是核心功能。每次打完牌，花半分钟录一场：

1. **选四位玩家**：从你的玩家库中勾选包括自己在内的 4 人。
2. **分配座位**：在 3×3 的方位盘中，点击东南西北四正位，为每个位置分配一位玩家。四个角位（东南、东北、西南、西北）会亮起当日财神方位的标记，方便对照。
3. **输入金额**：按东南西北顺序，分别输入每人这场的输赢。赢写正数，输写负数。页面上有 ±10 的快捷调整按钮。
4. **自动校验**：四人金额合计必须为 0（零和），满足后保存按钮才会出现，防止录错。

---

## 历史记录

- 按月翻页浏览所有过往场次。
- 每条记录显示日期、局数、你和其余三人的输赢金额。
- 点击记录可进入详情，支持**编辑**和**删除**。

---

## 数据统计

三个维度的可视化分析：

- **累计输赢趋势图**：折线图展示你的盈亏曲线变化，一目了然长期走势。
- **各方位胜率**：柱状图对比你坐东/南/西/北时的胜率差异，鼠标悬停可看对应方位场数和总输赢。
- **玩家对手统计**：列出每位牌友——同桌几场、你对他们时的胜率、你累计输赢、他们各自累计输赢。
- **上下家分析**：统计你的上家、下家、对家分别是哪些人时你的表现如何——谁坐你上家最旺你，谁坐你下家最克你。

---

## 智能座位推荐（亮点）

这是最有实战价值的页面：

1. **选择今日的三位对手**。
2. 系统自动找出历史上你和这三人同桌的所有场次。
3. 分别计算你坐东、南、西、北时的胜率和平均收益。
4. **推荐你坐哪个方位胜率最高**，并标注该方位是否恰好也是今日财神方位（双重利好）。

推荐结果附带完整的四方对比表格，包含场数、胜率、累计、平均收益，以及当日财位标记 ⭐。

## 我的（设置）

- **玩家库管理**：添加、改名、删除常打牌的牌友（删除前会检查历史关联，避免误删）。
- **干支校准**：如果应用计算的天干地支与你查看的黄历不一致，可在此调整基准日期和对应的干支，全局自动重新推算。
- **数据备份**：
  - **导出**：一键下载 JSON 文件，可存微信/电脑备份。
  - **导入**：换手机后一键恢复全部数据。
  - **清空**：彻底清除所有记录，需双重确认防误触。

---

## 使用技巧

- **iPhone 装到桌面**：Safari 打开 → 底部分享按钮 → "添加到主屏幕" → 命名"麻将记分"。之后点桌面图标即全屏打开，体验等同原生 App。
- **起步**：首次使用先去"我的"页添加几位常打牌的牌友，不然录入时选不到人。
- **干支校准**：基准默认锚定 2026-05-06 戊辰日，如果当天黄历不同，去"我的"页改一下即可。

---

## 技术说明

- 所有数据存于手机本地（localStorage），不会上传任何服务器。
- 首次联网加载 Chart.js（图表库），之后可离线使用，图表加载失败时自动降级为文本显示。
- 支持 iOS Safari PWA 全屏模式，包含自定图标和启动样式。
